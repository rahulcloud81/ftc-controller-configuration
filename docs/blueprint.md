# **App Name**: Controller Configurator

## Core Features:

- Visual Controller Display: Display an interactive image of the Xbox controller with visual indicators for assigned actions.
- Input Assignment: Allow users to assign robot commands to specific buttons, axes, and triggers on the controller through interactive fields.
- Mapping Profile Save: Enable users to save the current controller mapping configuration as a profile.
- Mapping Profile Load: Allow users to load previously saved controller mapping profiles.
- Configuration Print: Provide a print function to generate a readable document of the current controller configuration for easy reference.

## Style Guidelines:

- Primary color: Xbox Green (#107C10) to reflect the Xbox theme.
- Background color: Dark Gray (#242526) to complement the green and provide a neutral backdrop.
- Accent color: Light Gray (#B8B8B8) for interactive elements and highlights to ensure contrast.
- Body and headline font: 'Inter', sans-serif, to ensure clear readability.
- Use simple, clear icons to represent robot commands and actions.
- Organize the layout to have the controller image prominently displayed, with assignment fields located logically around it.
- Subtle animations to provide visual feedback on user interactions, such as button presses and profile loading.
