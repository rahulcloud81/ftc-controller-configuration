# Controller Configurator

This is a Next.js application for mapping Xbox controller inputs to custom robot actions.

## Features

- **Visual Controller Display**: An interactive SVG of an Xbox controller that highlights the selected input.
- **Input Assignment**: Assign custom string commands to each button, trigger, and axis.
- **Save/Load Profiles**: Save your current mapping configuration to a `.json` file and load it back later.
- **Print Configuration**: Generate a clean, printable summary of your current mappings for quick reference.

## Getting Started

To get started, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
